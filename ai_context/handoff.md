### How to run automated extraction

```bash
cd automation
python -m persona_extraction "我和女帝的九世孽缘" -r ..
```

Optional: `pip install jsonschema` for programmatic schema validation (the
tool works without it but skips schema checks).

The pipeline will:
1. Ask the LLM to analyze the book and produce a batch plan + candidate list
2. Show candidates interactively — user selects characters, confirms batch plan (generated by story boundaries), sets extraction range
3. Start the extraction loop (per-batch: extract → validate → review → commit)

If a progress file already exists (`works/我和女帝的九世孽缘/analysis/
incremental/extraction_progress.json`), the script will offer to resume.
You can also force resume with `--resume`.

See `automation/README.md` for full CLI options.
