# Persona Engine

Persona Engine is a multi-character roleplay architecture for long-form novels.

The project is designed around four persistent layers:

- source corpus
- character packages
- user relationship memory
- runtime compilation and terminal adapters

The current repository state is an architecture scaffold. The first goal is to
make the data model stable enough that future AI sessions, scripts, and product
surfaces can all build on the same package format.
