# 手动 Prompt 库

这个目录存放"可以直接发给一个全新的、没有项目背景的 AI agent"的固定提示词模板。

**仅保留手动场景使用的模板。** 自动化提取流程的 prompt 已迁入
`automation/prompt_templates/`；运行时 LLM 行为规则已迁入
`simulation/prompt_templates/`。

## 使用方式

1. 选择最接近当前任务的模板文件。
2. 把文件里的"可直接使用的提示词"整段发给新的 agent。
3. 把占位符替换成真实路径、书名、角色名等。
4. 如有必要，再把少量附加约束补在 prompt 末尾。

## 命名语言说明

本目录下的 prompt 文件和子目录均使用中文命名。这与 `ai_context/` 使用英文
的策略不冲突：

- `ai_context/` 面向 AI agent 的跨语言交接，使用英文
- `prompts/` 面向项目维护者和 AI agent 的工作流模板，使用中文
- JSON schema 的字段名保持英文

新 agent 在写任何目录或结构化文件前，应先回看
[最小结构读取入口](./shared/最小结构读取入口.md)。
