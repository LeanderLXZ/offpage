# 2026-04-01 17:40:25 - World-First Batch Order And Cross-Asset Updates

## Summary

The recommended extraction order was clarified:

1. identify candidate characters
2. extract shared world information in batches
3. extract one selected character in batches

This means deep character construction should normally happen after a usable
shared world base has been established for the work.

## Important Update Rule

Any one source-reading batch may update more than its immediate focus.

A batch may:

- add new world information
- revise previously written world information
- add new target-character information
- revise previously written target-character information
- add or revise information for other already-known characters

The key constraint is that these cross-asset updates must remain grounded in
source evidence rather than user dialogue or runtime drift.
